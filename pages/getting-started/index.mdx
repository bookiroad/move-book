# 🚀 시작하기 (Getting Started)

Movement Network에서 개발을 시작하는 방법을 단계별로 알아보겠습니다. 개발 환경 설정부터 첫 번째 Move 모듈 배포까지 차근차근 진행해보세요.

## 🎯 학습 목표

이 섹션을 완료하면:
- Movement 개발에 필요한 모든 도구를 설치할 수 있습니다
- 첫 번째 Move 모듈을 작성하고 컴파일할 수 있습니다  
- 테스트넷에서 계정을 생성하고 토큰을 받을 수 있습니다
- Movement CLI의 핵심 명령어를 사용할 수 있습니다
- 스마트 계약을 빌드하고 배포할 수 있습니다

## 📖 시작하기 가이드

### 1. [개발 환경 세팅](/getting-started/environment-setup)
Move 개발에 필요한 기본 도구들을 설치합니다:
- **Rust 설치** - Move 컴파일러의 기반
- **Git 설치** - 버전 관리를 위한 필수 도구
- **Movement CLI 설치** - 핵심 개발 도구
- **VS Code 확장** - 개발 효율성 향상

### 2. [첫 Move 모듈 만들기](/getting-started/first-module)
Hello World부터 시작하는 Move 개발:
- 새 프로젝트 생성
- 기본 모듈 구조 이해
- 간단한 함수 작성
- 컴파일과 테스트

### 3. [계정 생성 & Faucet 사용법](/getting-started/accounts-faucet)
Movement 네트워크에서 계정 관리:
- 테스트넷 계정 생성
- Faucet에서 테스트 토큰 받기
- 계정 정보 확인
- 잔액 조회

### 4. [Movement CLI 주요 명령어](/getting-started/cli-commands)
개발에 필수적인 CLI 명령어들:
- 프로젝트 관리 명령어
- 컴파일과 테스트 명령어
- 계정 관리 명령어
- 배포 관련 명령어

### 5. [Build & Deploy 기초 가이드](/getting-started/build-deploy)
스마트 계약을 실제 네트워크에 배포:
- Move.toml 설정 파일 이해
- 테스트넷에 배포하기
- 배포된 모듈 확인
- 함수 호출 테스트

## ⚡ 빠른 시작

바로 시작하고 싶다면 다음 명령어들을 순서대로 실행해보세요:

### 1. 개발 환경 확인
```bash
# Rust 설치 확인
rustc --version

# Git 설치 확인  
git --version

# Movement CLI 설치 확인
movement --version
```

### 2. 첫 프로젝트 생성
```bash
# 새 Move 프로젝트 생성
movement move init hello-movement
cd hello-movement

# 프로젝트 구조 확인
ls -la
```

### 3. 기본 빌드 테스트
```bash
# 프로젝트 빌드
movement move build

# 테스트 실행
movement move test
```

## 💡 사전 요구사항

Movement 개발을 시작하기 전에 다음 사항들을 확인해주세요:

### 운영체제 지원
- **macOS**: 10.15 이상
- **Linux**: Ubuntu 18.04+ 또는 동등한 배포판
- **Windows**: WSL2 권장 (Windows 10/11)

### 기본 지식
- **터미널/명령줄 사용법**: 기본적인 bash 명령어
- **텍스트 에디터 사용법**: VS Code, Vim 등
- **Git 기초**: clone, commit, push 정도면 충분

### 권장 사양
- **RAM**: 4GB 이상 (8GB 권장)
- **저장공간**: 2GB 이상 여유 공간
- **인터넷 연결**: 패키지 다운로드 및 테스트넷 접속용

## 🛠️ 개발 도구 미리보기

Movement 개발에 사용할 주요 도구들:

### Movement CLI
```bash
# 프로젝트 생성
movement move init my-project

# 빌드
movement move build

# 테스트
movement move test

# 배포
movement move publish
```

### VS Code 확장
- **Move Language Extension**: 문법 하이라이팅, 자동완성
- **Move Analyzer**: 실시간 에러 체크, 코드 분석

## 🌐 네트워크 정보

Movement는 여러 네트워크를 제공합니다:

### Mainnet (메인넷)
- **RPC**: `https://mainnet.movementnetwork.xyz/v1`
- **Explorer**: `https://explorer.movementnetwork.xyz/?network=mainnet`
- **용도**: 실제 서비스 배포

### Bardock Testnet (테스트넷)
- **RPC**: `https://testnet.bardock.movementnetwork.xyz/v1`
- **Faucet**: `https://faucet.movementnetwork.xyz/`
- **Explorer**: `https://explorer.movementnetwork.xyz/?network=bardock+testnet`
- **용도**: 개발 및 테스트

### Devnet (개발넷)
- **RPC**: `https://devnet.suzuka.movementnetwork.xyz/v1`
- **Faucet**: `https://faucet.devnet.suzuka.movementnetwork.xyz/`
- **용도**: 실험적 기능 테스트
