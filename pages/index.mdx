# Move Book

**Move Book**은 Movement Network의 한국어 개발자 문서입니다.

Move 언어부터 dApp 개발까지 체계적으로 학습할 수 있는 완전한 가이드를 제공합니다.

## 🚀 시작하기

Movement Network 개발을 시작하는 방법을 알아보세요:

- 📚 **기본 개념**: Move 언어의 핵심 개념들을 학습합니다
- 🎯 **고급 개념**: Smart Vector, Smart Table, Resource Account를 다룹니다  
- 🏗️ **dApp 개발**: NFT, DeFi, 게임 개발을 실습합니다

## 💡 특징

- **한국어 완전 지원**: 영어 장벽 없이 학습 가능
- **실전 중심**: 바로 사용할 수 있는 예제와 코드
- **체계적 구성**: 기초부터 고급까지 단계별 학습
- **해커톤 최적화**: 빠른 프로토타입 개발 지원
