# Movement Move Book

**Movement에서 시작하는 Move 개발 가이드**

Move 언어부터 dApp 개발까지 체계적으로 학습할 수 있는 완전한 가이드를 제공합니다.

- Movement CLI 설치 및 설정
- VS Code/Cursor 확장 프로그램 설정
- 첫 Move 모듈 만들기
- 계정 생성 & Faucet 사용법
- Movement CLI 주요 명령어
- Build & Deploy 기초 가이드

### 🔧 **[개발자 도구](/developer-tools)**

- Full Node 운영 가이드

- 제어 구조 & 함수 타입

## 🛠️ 빠른 시작

```bash
# Movement CLI 설치
curl -fsSL https://get.movementnetwork.xyz | sh

# 새 프로젝트 생성
movement init my-project
cd my-project

# 개발 서버 시작
movement dev
```
